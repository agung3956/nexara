Nexara TaskOS - Dashboard Monitoring Tugas

Cara pakai:
1. Buka file index.html di browser.
2. Klik Tambah Tugas untuk membuat tugas baru.
3. Klik kartu tugas untuk edit detail.
4. Checklist bisa dicentang.
5. Data tersimpan otomatis di localStorage browser.

Catatan:
- Aplikasi ini adalah versi demo frontend single-file.
- Tidak membutuhkan server.
- Untuk versi production, gunakan Next.js, Supabase/PostgreSQL, authentication, RBAC, dan audit trail.
